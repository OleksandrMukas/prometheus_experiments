# prometheus_experiments
Testing and labs about Prometheus

This script tested on Oracle Linux 10.1

9090 port has Oracle Cockpit on default. We can delete or stop+disable this service on host.

Or run Prometheus on enother port - 9091 or 9092 etc.

###BIN     -> /usr/local/bin/prometheus
###CONF    -> /etc/prometheus/prometheus.yml
###DATA    -> /var/lib/prometheus/data
